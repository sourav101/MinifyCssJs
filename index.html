Welcome

Welcome to simple css js minifier. You can easily use simple css js minifier to minify and combine your css or js files.

This minifier also can combine your css js files and its provide you a single manifest file.

It also able to acess directory and sub directory css/js file from input directory.

By using this single file your website will be more faster.

Thank you for using this minifest service

How to use

You have to follow some step.
Step 1: First download it.
Step 2: Extract it and move this file to your apache server(localhost).
Step 3: Put all of your css/js files to input directory.
Step 4: Now you have to run index.php file and select an option (Css/Js) and then click on Minify button.
Ok done!
Finally you have successfully got a manifest file in the output directory.
Now you are able to use this file.
